Aloha!

/svn is svn server, /git is git server.

You will need an account to access them.

To create a svn repo, ssh into aloha, run 

$ sudo svnadmin create /var/svn/test
$ chown -R www-data:www-data /var/svn/test
$ chmod -R 775 /var/svn/test

To create a git repo, ssh into aloha, run 

$ sudo git-create-repo.sh test
Git repository 'test' created in /var/git/test.git
